# 貝爾的FFXIV市場小屋

> 專為 Final Fantasy XIV 玩家打造的現代化市場價格比較工具..

**作者：** 貝爾 Beher 

---

## ✨ 功能特色

### 🔍 **智能物品搜尋**
- 支援**繁體中文**或**簡體中文**物品名稱搜尋
- 自動轉換繁體與簡體中文
- 即時搜尋結果
- 支援部分匹配與多關鍵字搜尋

### 💰 **市場價格比較**
- 跨多個伺服器與資料中心比較價格
- 查看詳細市場掛單（支援HQ/NQ選項）
- 追蹤近期購買歷史與趨勢
- 僅顯示高品質（HQ）物品篩選
- 顯示最低價格、平均價格與流通速度數據

### 🌳 **製作樹狀圖**
- 視覺化完整製作配方與依賴關係
- 一目了然查看材料成本
- 追蹤製作鏈中所有材料的價格
- 互動式樹狀圖導航

### 🛡️ **主線裝備查價器**
- 依物品等級範圍搜尋裝備
- 依裝備部位篩選（武器、防具、飾品）
- 適合尋找經濟實惠的主線進度裝備
- 跨伺服器比較價格

### ⚔️ **製作職業裝備搜尋器**
- 尋找特定製作職業的最佳價格裝備
- 依物品等級範圍篩選
- 可選擇多個職業進行比較
- 優化您的製作裝備採購

### 📊 **其他工具**
- **相關物品**：探索與您搜尋相關的物品
- **搜尋歷史**：追蹤您的近期搜尋記錄
- **物品歷史**：快速存取先前查看過的物品
- **外部連結**：快速連結至 Garland Tools 與 FFXIV Wiki

### 🎨 **使用者體驗**
- 精美的 FFXIV 主題深色介面
- 完全響應式設計（桌面、平板、手機）
- 快速且輕量
- 即時數據更新

## 🚀 快速開始

1. **搜尋物品** - 輸入繁體或簡體中文物品名稱
2. **選擇物品** - 從搜尋結果中點選物品
3. **選擇伺服器** - 選擇一個或多個伺服器/資料中心進行比較
4. **查看價格** - 查看詳細市場數據，包括掛單、歷史與統計資料

### 進階功能

- **製作樹狀圖**：點選有製作配方的物品以查看完整製作樹
- **主線裝備查價器**：前往 `/msq-price-checker` 尋找經濟實惠的主線裝備
- **職業裝備搜尋器**：前往 `/ultimate-price-king` 尋找製作職業裝備
- **歷史記錄**：前往 `/history` 查看您的搜尋與物品歷史

### ⚠️ 重要提示

- **後端更新**：目前新版本會需要作者手動更新後端，如有功能異常請聯繫作者DC：wuperbear

## 📖 使用範例

### 範例一：尋找市場價格
```
1. 搜尋「風化」或「风化」（風化物品）
2. 從結果中選擇物品
3. 選擇伺服器，例如「陸行鳥」或特定世界
4. 比較價格並找到最佳交易
```

### 範例二：檢查製作成本
```
1. 搜尋可製作物品
2. 點選製作樹狀圖圖示
3. 查看所有所需材料及其價格
4. 計算總製作成本
```

### 範例三：尋找主線裝備
```
1. 前往主線裝備查價器
2. 設定物品等級範圍（例如：50-60）
3. 選擇裝備部位（例如：「主手武器」）
4. 跨伺服器比較價格
```

## 🛠️ 開發者資訊

### 技術堆疊

- **React** - 現代化 UI 框架
- **Vite** - 快速建置工具與開發伺服器
- **Tailwind CSS** - 工具優先的 CSS 框架
- **React Router** - 客戶端路由

### API 與資料來源

- **[Universalis API](https://universalis.app/)** - 即時市場看板數據
- **[XIVAPI](https://xivapi.com/)** - 物品圖片與圖示
- **[thewakingsands/ffxiv-datamining-cn](https://github.com/thewakingsands/ffxiv-datamining-cn)** - 簡體中文物品數據

### 開發環境設定

```bash
# 安裝依賴套件
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 預覽生產版本
npm run preview
```

### 專案結構

```
src/
├── components/     # React 元件
├── services/       # API 服務與資料處理
├── utils/          # 工具函數
├── hooks/          # 自訂 React Hooks
└── styles/         # CSS 樣式
```
